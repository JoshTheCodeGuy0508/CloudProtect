This app is still in active development and will not work as described if run in its current state!

The concept of this app is to avoid the obvious privacy issues that arise when uploading our sensitive data to the cloud.
Development of initial functionality targets Google Drive, with more cloud providers expected after the first full release of this app.
Suggestions are welcome as well as information on any bugs you may notice in the code.
Be sure to star this repository so you do not miss the official release of CloudProtect v. 1.0!
